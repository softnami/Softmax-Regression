Parameters:

  • args Object

    These are the required arguments.

    • learningRate Number

      Learning rate for BackPropogation.

    • threshold Number

      Threshold value for cost.

    • regularization_parameter Number

      Regularization parameter to prevent overfitting.

    • notify_count Number

      Value to execute the iteration_callback after every x number of epochs.

    • iteration_callback Function

      Callback that can be used for getting cost and epoch value on every notify count.

    • maximum_epochs Number

      Maximum epochs to be allowed before the optimization is complete. Defaults to 1000.

    • weight_initilization_range Array

      Array contains the lower bound and upper bound for random weight initilization: [lower bound, upper bound].

    • parameter_size Array

      Array contains the number of input features and total number of output classes: [number of input features, total number of output classes].
